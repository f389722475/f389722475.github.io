
本网站使用 **Astro** 框架和 [Mizuki](https://github.com/matsuzaka-yuki/mizuki) 主题构建。

::github{repo="matsuzaka-yuki/Mizuki"}

## 🌟 主题特色

### 🎨 设计与用户体验
- **现代优雅的设计** - 简洁的极简界面与精美的排版
- **全面响应式布局** - 针对从移动端到桌面端的所有设备进行优化
- **深色/浅色模式** - 自动切换主题，并提供流畅的过渡效果
- **精美排版** - 使用 JetBrains Mono 字体提升可读性
- **流畅动画** - 顺滑的页面切换与交互效果

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 提供支持，快速呈现精准结果
- **增强型 Markdown** - 支持扩展语法、代码高亮与数学公式
- **交互式目录** - 为长文章提供自动滚动导航
- **RSS 订阅源生成** - 自动生成订阅源，方便及时获取更新
- **阅读时长估算** - 预先了解阅读文章所需的时间
- **文章分类** - 使用标签与分类整理内容



### 📱 特殊页面
- **追番页面** - 记录动画观看进度与评分
- **友链页面** - 使用精美卡片展示朋友的网站
- **日记/动态页面** - 像发布社交媒体动态一样分享生活瞬间
- **归档页面** - 以清晰的时间线展示所有文章
- **关于页面** - 可自定义的个人介绍（就是本页！）

### 🛠 技术特性
- **增强型代码块** - 由 [Expressive Code](https://expressive-code.com/) 提供支持
- **数学公式支持** - 使用 KaTeX 渲染 LaTeX 数学表达式
- **图片优化** - 支持延迟加载的 PhotoSwipe 图库
- **SEO 优化** - 内置站点地图与元标签，提升搜索可见度
- **性能优化** - 通过缓存与优化实现快速加载
- **评论系统支持** - 支持集成 Twikoo 评论系统

### 🎯 高级 Markdown 功能
- **提示框与警告框** - 使用 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]` 创建精美的信息框
- **数学公式** - 使用 `$inline$` 与 `$$block$$` 语法编写 LaTeX 数学公式
- **GitHub 卡片** - 使用 `::github{repo="user/repo"}` 嵌入仓库卡片
- **语法高亮** - 支持行号的高级代码高亮
- **复制代码按钮** - 轻松复制代码
